# ShopSense Documentation Website

This repository contains the documentation site for **ShopSense**, a mobile app that helps users remember what to buy at the right place and time through simple, location-based shopping reminders.

The website explains:
- How the app works  
- How to manage the to-buy list  
- How to set a home location  
- How to use Shopping Mode  
- What happens behind the scenes  

The guide is designed with a clean layout to make the instructions easy to follow.
